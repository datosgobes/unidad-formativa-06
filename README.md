Recursos didácticos abiertos sobre **Reutilización de la Información del Sector Público (RISP)**, centrados en el perfil de aplicación español **DCAT-AP** y la **NTI-RISP**. Materiales interactivos creados con LiaScript para una experiencia formativa completa.

## Contenido del curso

Esta unidad formativa aborda:

- **DCAT-AP**: Perfil de aplicación del estándar europeo para la descripción de catálogos de datos
- **NTI-RISP**: Norma Técnica de Interoperabilidad de Reutilización de Recursos de Información
- Implementación práctica de metadatos para catálogos de datos abiertos
- Interoperabilidad y buenas prácticas en la publicación de información del sector público

## Uso y desarrollo

### Ver el curso localmente

```bash
# Instalar el servidor de desarrollo de LiaScript
npm install -g @liascript/devserver

# Clonar el repositorio
git clone https://github.com/datosgobes/unidad-formativa-06.git
cd unidad-formativa-06

# Iniciar el servidor local
liascript-devserver --open
```

### Editar el contenido

1. El contenido principal está en [CURSO.md](CURSO.md)
2. Puedes usar el [LiveEditor de LiaScript](https://liascript.github.io/liveeditor/) para editar en línea
3. O editar localmente con tu editor preferido y previsualizar con `liascript-devserver`
